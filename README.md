# Peer Sphere

**Group 7** | **Group Leader:** Divine Oke

**Theme:** An Interconnected world

A digital ecosystem inspired by Gerald Chukwuma's *"A World Without War"* where students from every corner of the globe connect seamlessly — breaking down geographical, cultural, and academic barriers to build a truly interconnected learning community.

---

## Tech Stack

- **Language:** Python
- **Design:** Figma
- **GUI Framework:** Tkinter
- **Data Storage:** CSV file for user authentication and profile storage
- **Code Storage:** GitHub
- **MS Excel:** CSV File
- **MS Word:** Product Summary Document

---

## Problem Statement

Students around the world face significant barriers to cross-border collaboration and connectivity. Many lack access to a centralized, easy-to-use platform where they can connect with peers from different cultures for academic support, project collaboration, and cultural exchange. Existing solutions are often either too complex (requiring steep learning curves), too fragmented (spread across multiple apps), or too impersonal.
